# Fundamentos de la Programación

Repositorio con ejemplos y ejercicios en Python para el ramo de **Fundamentos de la Programación**.

---

## Get Started

### Requisitos previos

- Tener instalado [Git](https://git-scm.com/)
- Tener instalado [Python](https://www.python.org/) (versión 3.x recomendada)

### Clonar el repositorio

```bash
git clone https://github.com/amaurelia/fundamentos.git
```

## Contenido

### 📚 Material de estudio

Carpeta con ejemplos listos para revisar y ejecutar, organizados por tema:

- **Variables** — declaración y tipos de datos
- **Entrada / Salida** — interacción con el usuario
- **Condicionales** — toma de decisiones con `if`, `elif`, `else`
- **Ciclos** — repetición con `for` y `while`
- **Match** — estructura de control `match`
- **Menú** — construcción de menús en consola
- **Excepciones** — manejo de errores con `try/except`

### 🏋️ Ejercicios

Tres carpetas con ejercicios propuestos para que los alumnos practiquen los conceptos estudiados:

- **Ejercicios Menú** — programas con menús interactivos (avistamientos, banco, cantante, combo, libros, promedio, auto)
- **Ejercicios Descuentos** — cálculo de descuentos con códigos, cantidad y notas (ropa, cine, cursos)
- **Ejercicios Random** — uso de números aleatorios (dados, personajes, acciones, lotería)
